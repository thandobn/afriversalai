Create a user

The admin-create-user Edge Function isn’t deployed yet. Deploy it (see supabase/functions/admin-create-user) to create accounts here.

Create any account type. Choose Partner and set a password to add a partner with login credentials and full Partner Portal privileges — they can sign in straight away. Leave the password blank to email a secure set-password link instead.

Set a user’s password

The admin-set-password Edge Function isn’t deployed yet, so this can’t set passwords. Deploy it (see supabase/functions/admin-set-password) to enable.

Set a new password for any account — learner, partner or facilitator — by email. Useful when someone is locked out and reset emails aren’t reaching them. Share the temporary password and ask them to change it after signing in.

📧 Email system (Resend)

The system emails registered users automatically (welcome on sign-up, partner invites, cohort access codes) through Resend. Send a test email here to confirm the integration is live.

Not sending yet? The send-email function needs to be deployed with a Resend API key and a verified afriversal.ai domain — see supabase/functions/send-email.
